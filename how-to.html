How to Create a Quiz

There are two main ways to create a quiz: using the AI Generator or manually creating a text file.

Method 1: AI Generator (Recommended)

You can automatically generate a quiz by providing a document and your own OpenAI API key.

  1. Get an OpenAI API Key: You will need an API key from OpenAI. You can get one by signing up at platform.openai.com. After signing up, go to the "API Keys" section in your account to create a new secret key.
  2. Note on Billing: Using the API is not free. You must set up a billing method in your OpenAI account. However, usage is very cheap (a few cents for many quizzes).
  3. Go to the Creator Page: Navigate back to the main page and find the "Generate with AI" section.
  4. Paste Your Key & Upload: Paste your secret API key into the provided input box and upload your document (.txt, .pdf, etc.). The website will then generate the quiz for you.

Method 2: Manual File Creation

If you prefer not to use an API key, you can use an AI chatbot with the prompt below to generate the file content, then save it as a `.txt` file and upload it.

You are a highly precise file conversion utility. Your only function is to convert the provided text from a question paper into a specific 7-line format. Your entire output MUST be ONLY the formatted text content. Do not write any other words. For any mathematical notation, enclose it in LaTeX delimiters like \(...\). If a question in the source does NOT have a clear, single letter (A, B, C, or D) answer, OMIT that entire question.

## REQUIRED FORMAT
[The full question text]
[The text for option A]
[The text for option B]
[The text for option C]
[The text for option D]
ANSWER: [The single, correct lowercase letter]
EXPLANATION: [An explanation, or blank]
